# GitHub Search

A React app that searches the GitHub API by keyword across multiple result types — repos, commits, issues, topics, and users.

**Live demo:** https://merry-parfait-883a1b.netlify.app/

## Features

- Search GitHub by keyword across multiple types (repos, commits, issues, topics, users)
- Results displayed in dynamic cards, adapting to the varying shape of each result type

## Tech Stack

- React, JavaScript
- GitHub REST API

## Getting Started

```bash
npm run dev    # development
npm run build  # production build
```
